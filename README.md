# Selenium Java Examples

A collection of Selenium WebDriver examples implemented using Java. This repository demonstrates core Selenium concepts with well-organized examples, covering browser operations, web elements, synchronization, user interactions, screenshots, and Excel file handling.

---

## Technologies Used

- Java 21
- Selenium WebDriver 4.44.0
- Apache POI 5.2.2
- Apache Commons IO
- Eclipse IDE
- Git & GitHub

## How to Run

1. Clone the repository.
2. Open the project in Eclipse.
3. Add the required Selenium and Apache POI libraries to the Build Path.
4. Run any Java class as a Java Application.

## Repository Purpose

This repository is intended for learning Selenium WebDriver concepts through simple, independent examples. It serves as a reference while learning automation testing with Java.
